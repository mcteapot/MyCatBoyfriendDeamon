# Story Structure Architect Skill

## Skill Name
Story Structure Architect

## Purpose
This skill helps an AI writing agent understand, compare, select, and apply popular fiction story structures for novels, screenplays, manga, anime, webnovels, vertical dramas, microdramas, games, and serialized fiction.

The skill should be used when the user wants to:
- Develop a story outline
- Choose a plot structure
- Convert an idea into a beat sheet
- Analyze an existing story
- Build a screenplay, manga arc, novel, webtoon, or microdrama
- Improve pacing, conflict, emotional escalation, or character transformation
- Combine multiple storytelling formulas into a stronger commercial structure

The agent should not treat any structure as a rigid rule. Story structures are tools. The best result often comes from combining several frameworks.

---

# Core Operating Principles

## 1. Always Identify the Story Type First
Before applying a structure, determine the story's format and genre.

Ask or infer:
- Is it a feature film, novel, manga, anime, series, microdrama, short, game, or webtoon?
- Is the genre romance, fantasy, mystery, horror, comedy, thriller, action, slice of life, tragedy, or melodrama?
- Is the story character-driven, plot-driven, relationship-driven, or mystery-driven?
- Is the tone commercial, literary, comedic, dark, epic, cozy, melodramatic, or serialized?

## 2. Match Structure to Story Function
Each structure solves a different problem.

Use:
- Hero's Journey for transformation and adventure
- Save the Cat for commercial screenplay pacing
- Three-Act Structure for broad story organization
- Dan Harmon's Story Circle for episodic transformation
- Mystery Formula for investigations and reveals
- Kishotenketsu for twist-based or non-conflict-driven stories
- Romance Structure for emotional relationship arcs
- Tragedy Structure for downfall stories
- Shonen Battle Manga Structure for growth, rivals, and escalating battles
- Soap Opera / Vertical Drama Formula for emotional cliffhangers
- Seven-Point Structure for novels and fantasy plotting
- Fichtean Curve for thrillers, horror, and fast-paced serials

## 3. Combine Structures When Useful
For commercial fiction, combine frameworks.

Examples:
- Fantasy romance: Hero's Journey + Romance Structure + Save the Cat
- Mystery thriller: Mystery Formula + Fichtean Curve + Three-Act Structure
- Microdrama: Soap Opera Formula + Romance Structure + Fichtean Curve
- Shonen manga: Hero's Journey + Shonen Battle Structure + Story Circle
- Slice of life anime: Kishotenketsu + Story Circle
- Tragic romance: Romance Structure + Tragedy Structure

## 4. Prioritize Emotional Escalation
Good fiction usually escalates in at least one of these areas:
- Stakes
- Secrets
- Danger
- Desire
- Betrayal
- Power
- Mystery
- Romantic tension
- Moral consequences
- Identity revelation

## 5. Make Every Beat Do Work
Each beat should ideally include:
- A clear event
- A character choice
- A consequence
- An emotional turn
- A new question or complication

# Story Structure Library

## 1. The Hero's Journey

### Best For
Fantasy, adventure, superhero stories, shonen anime, chosen-one stories, mythic drama, quest narratives.

### Core Idea
A character leaves their ordinary world, enters a dangerous unknown world, is tested, transforms, and returns changed.

### Common 12 Beats
1. Ordinary World
2. Call to Adventure
3. Refusal of the Call
4. Meeting the Mentor
5. Crossing the Threshold
6. Tests, Allies, and Enemies
7. Approach to the Inmost Cave
8. Ordeal
9. Reward
10. Road Back
11. Resurrection
12. Return with the Elixir

### Use When
The story is about personal transformation through danger, adventure, and trial.

### Agent Application
When applying Hero's Journey:
- Define the hero's starting flaw or limitation
- Define the unknown world they must enter
- Define the mentor, threshold, ordeal, and transformation
- Make the final victory prove the hero has changed

### Output Template
```markdown
## Hero's Journey Outline

### Ordinary World
[Hero's normal life before the adventure.]

### Call to Adventure
[The event that disrupts normal life.]

### Refusal of the Call
[Why the hero hesitates.]

### Mentor
[Who guides or challenges them.]

### Crossing the Threshold
[The moment the hero cannot go back.]

### Tests, Allies, Enemies
[Early trials and key relationships.]

### Approach
[The hero nears the central danger.]

### Ordeal
[The major crisis or symbolic death.]

### Reward
[What the hero gains.]

### Road Back
[The consequences of victory begin.]

### Resurrection
[Final test proving transformation.]

### Return with the Elixir
[How the hero returns changed.]
```

## 2. Save the Cat

### Best For
Commercial screenplays, feature films, romantic comedies, family films, genre movies, vertical drama season planning.

### Core Idea
A story is built around 15 emotional and structural beats that guide the audience from setup to transformation.

### 15 Beats
1. Opening Image
2. Theme Stated
3. Setup
4. Catalyst
5. Debate
6. Break into Act 2
7. B Story
8. Fun and Games
9. Midpoint
10. Bad Guys Close In
11. All Is Lost
12. Dark Night of the Soul
13. Break into Act 3
14. Finale
15. Final Image

### Use When
The user wants a clean commercial outline with strong pacing.

### Agent Application
When applying Save the Cat:
- Create a strong contrast between Opening Image and Final Image
- Make the Catalyst force change
- Use the Midpoint as either a false victory or false defeat
- Make All Is Lost emotionally painful
- Make the Finale answer the theme

### Output Template
```markdown
## Save the Cat Beat Sheet

### Opening Image
[Visual snapshot of the hero's life before change.]

### Theme Stated
[The story's central lesson or emotional question.]

### Setup
[World, hero, flaw, relationships, problem.]

### Catalyst
[The event that changes everything.]

### Debate
[The hero hesitates, questions, or resists.]

### Break into Act 2
[The hero enters a new world or situation.]

### B Story
[Relationship or emotional subplot.]

### Fun and Games
[The promise of the premise.]

### Midpoint
[False victory or false defeat; stakes rise.]

### Bad Guys Close In
[External and internal pressure increases.]

### All Is Lost
[The hero's lowest point.]

### Dark Night of the Soul
[The hero processes the loss.]

### Break into Act 3
[The hero finds the solution.]

### Finale
[The hero acts with new understanding.]

### Final Image
[Visual proof that the hero has changed.]
```

## 3. Three-Act Structure

### Best For
Almost all fiction formats, especially screenplays, novels, and short stories.

### Core Idea
A story is divided into setup, confrontation, and resolution.

### Three Acts
1. Act 1: Setup
2. Act 2: Confrontation
3. Act 3: Resolution

### Common Beat Pattern
- Opening situation
- Inciting incident
- First turning point
- Rising complications
- Midpoint
- Crisis
- Climax
- Resolution

### Use When
The user needs a simple, universal structure.

### Agent Application
When applying Three-Act Structure:
- Act 1 should establish desire, flaw, world, and conflict
- Act 2 should escalate obstacles and force growth
- Act 3 should resolve the central conflict through decisive action

### Output Template
```markdown
## Three-Act Outline

### Act 1: Setup
[Introduce the world, protagonist, desire, flaw, and inciting incident.]

### Act 1 Turning Point
[The choice or event that launches the main story.]

### Act 2: Confrontation
[The protagonist pursues the goal and faces escalating obstacles.]

### Midpoint
[A major reversal, reveal, victory, or defeat.]

### Act 2 Crisis
[The worst consequence before the climax.]

### Act 3: Resolution
[The protagonist makes a final choice and confronts the central conflict.]

### Climax
[The decisive confrontation.]

### Resolution
[The new status quo.]
```

## 4. Dan Harmon's Story Circle

### Best For
TV episodes, short stories, animated episodes, games, webisodes, micro episodes, comedic or episodic storytelling.

### Core Idea
A character starts in comfort, wants something, enters an unfamiliar situation, gets what they want, pays a price, returns changed.

### 8 Steps
1. You
2. Need
3. Go
4. Search
5. Find
6. Take
7. Return
8. Change

### Use When
The user wants a compact episode or scene arc.

### Agent Application
When applying Story Circle:
- Define the character's comfort zone
- Define what they want
- Force them into unfamiliar territory
- Make them pay a price
- End with change, even if small

### Output Template
```markdown
## Story Circle Outline

### You
[Who the character is in their comfort zone.]

### Need
[What they want or lack.]

### Go
[How they enter an unfamiliar situation.]

### Search
[How they struggle and adapt.]

### Find
[What they discover or obtain.]

### Take
[The price they pay.]

### Return
[How they come back to a familiar world.]

### Change
[How they are different.]
```

## 5. 12-Step Mystery Formula

### Best For
Detective fiction, murder mystery, thriller, crime, supernatural mystery, investigative horror.

### Core Idea
A mystery begins with a question, introduces suspects and clues, misleads the audience, then reveals the hidden truth.

### 12 Steps
1. Crime or central mystery
2. Investigator enters
3. Suspects introduced
4. First clues discovered
5. Red herrings appear
6. Hidden motives emerge
7. Stakes increase
8. False solution
9. New evidence
10. Revelation
11. Confrontation
12. Explanation and resolution

### Use When
The story is powered by secrets, investigation, and revelation.

### Agent Application
When applying Mystery Formula:
- Define the central question
- Define the truth before writing clues
- Plant real clues and false clues
- Give each suspect motive, means, and opportunity
- Make the final reveal feel surprising but inevitable

### Output Template
```markdown
## Mystery Outline

### Central Mystery
[The question the story must answer.]

### The Truth
[What really happened.]

### Investigator
[Who seeks the answer and why.]

### Suspects
[List suspects with motive, means, and opportunity.]

### Clues
[Real clues that point toward the truth.]

### Red Herrings
[False clues that mislead the audience.]

### False Solution
[The wrong answer that seems convincing.]

### Revelation
[The key discovery that changes everything.]

### Confrontation
[The investigator faces the culprit or truth.]

### Explanation
[How all clues connect.]
```

## 6. Kishotenketsu

### Best For
Manga, anime, slice of life, literary fiction, gentle comedy, emotional shorts, twist-driven stories, non-Western narrative forms.

### Core Idea
A story can be built around introduction, development, twist, and reconciliation without relying on direct conflict.

### Four Parts
1. Ki: Introduction
2. Sho: Development
3. Ten: Twist
4. Ketsu: Conclusion

### Use When
The user wants a story based on contrast, surprise, meaning, or emotional discovery rather than conflict.

### Agent Application
When applying Kishotenketsu:
- Introduce a situation clearly
- Develop it naturally
- Add a twist that recontextualizes the situation
- Resolve by connecting the first three parts into meaning

### Output Template
```markdown
## Kishotenketsu Outline

### Ki: Introduction
[The situation, character, or world.]

### Sho: Development
[Expansion of the situation.]

### Ten: Twist
[A surprising contrast, reversal, or new context.]

### Ketsu: Conclusion
[How the twist changes the meaning of the story.]
```

## 7. Romance Structure

### Best For
Romance novels, romantic comedy, fantasy romance, vertical romance dramas, love triangles, relationship-centered plots.

### Core Idea
Two characters are drawn together, blocked by internal and external obstacles, separated by conflict, then reunited through emotional truth.

### Common Beats
1. Meet Cute or charged first encounter
2. Attraction or fascination
3. Forced proximity or repeated contact
4. Emotional vulnerability
5. First major romantic turn
6. First kiss or commitment signal
7. Deepening bond
8. Secret, lie, misunderstanding, or betrayal
9. Breakup or separation
10. Realization
11. Grand gesture or final proof of love
12. Reunion or emotional resolution

### Use When
The emotional engine is desire, intimacy, trust, and vulnerability.

### Agent Application
When applying Romance Structure:
- Define what each character wants
- Define what each character fears
- Make attraction specific
- Make the breakup come from the characters' wounds or secrets
- Make the reunion prove growth

### Output Template
```markdown
## Romance Beat Sheet

### First Encounter
[How the leads meet or collide.]

### Initial Attraction
[What pulls them toward each other.]

### Initial Resistance
[Why they cannot easily be together.]

### Forced Proximity
[Why they must keep interacting.]

### Vulnerability
[First emotional opening.]

### Romantic Turn
[Moment the relationship clearly changes.]

### Deepening Bond
[How intimacy grows.]

### Major Obstacle
[Secret, betrayal, misunderstanding, rival, or external force.]

### Breakup or Separation
[Why the relationship appears impossible.]

### Realization
[What each character understands.]

### Grand Gesture
[The action proving love or growth.]

### Resolution
[How the relationship ends or transforms.]
```

## 8. Tragedy Structure

### Best For
Dark drama, downfall stories, revenge stories, moral corruption arcs, cautionary tales.

### Core Idea
A character with potential is destroyed by a fatal flaw, bad choice, or corrupting desire.

### Common Beats
1. Noble or promising character
2. Fatal flaw introduced
3. Temptation or opportunity
4. Early success
5. Moral compromise
6. Warnings ignored
7. Consequences escalate
8. Point of no return
9. Recognition
10. Downfall

### Use When
The story is about destruction rather than triumph.

### Agent Application
When applying Tragedy Structure:
- Define the fatal flaw
- Give the hero chances to turn back
- Make each bad choice understandable
- Make the ending feel inevitable
- Include recognition before or during the downfall

### Output Template
```markdown
## Tragedy Outline

### Promise
[Who the character could have become.]

### Fatal Flaw
[The weakness that leads to ruin.]

### Temptation
[What activates the flaw.]

### Early Success
[How the flaw seems to help at first.]

### Moral Compromise
[The first meaningful wrong choice.]

### Warnings
[Who or what warns the character.]

### Escalation
[Consequences grow worse.]

### Point of No Return
[The choice that cannot be undone.]

### Recognition
[The character understands the truth.]

### Downfall
[The final consequence.]
```

## 9. Shonen Battle Manga Structure

### Best For
Battle manga, anime arcs, supernatural action, martial arts stories, power-growth stories, rival-driven fiction.

### Core Idea
The hero faces stronger opponents, trains, fails, grows, and wins through new power, friendship, strategy, or emotional resolve.

### Common Cycle
1. New goal or threat
2. Rival or enemy introduced
3. Training or skill discovery
4. First confrontation
5. Defeat or partial victory
6. Mentor lesson or power breakthrough
7. Team bonds tested
8. Rematch
9. Victory
10. Bigger threat revealed

### Use When
The story is driven by escalation, power systems, rivals, tournaments, missions, and emotional growth through combat.

### Agent Application
When applying Shonen Structure:
- Define the power system
- Define the hero's emotional weakness
- Define rival contrast
- Make every battle reveal character
- Use defeat as a growth mechanism
- End arcs by revealing a bigger world or threat

### Output Template
```markdown
## Shonen Battle Arc Outline

### New Threat or Goal
[What challenge appears.]

### Rival or Enemy
[Who opposes or contrasts the hero.]

### Power System Rule
[What ability, technique, or limitation matters.]

### Training or Discovery
[How the hero prepares or learns.]

### First Battle
[Initial confrontation and result.]

### Defeat or Setback
[What the hero lacks.]

### Breakthrough
[New technique, insight, or emotional growth.]

### Team Test
[How allies matter.]

### Rematch or Final Battle
[The decisive fight.]

### Victory and Reveal
[What is won and what larger threat appears next.]
```

## 10. Soap Opera / Vertical Drama Formula

### Best For
Microdramas, vertical dramas, melodrama, serialized romance, revenge stories, billionaire drama, hidden identity stories.

### Core Idea
Every episode creates an emotional hook, escalates conflict, reveals a secret, and ends with a cliffhanger.

### Episode Formula
1. Hook
2. Conflict
3. Revelation
4. Escalation
5. Cliffhanger

### Series-Level Ingredients
- Hidden identity
- Betrayal
- Secret marriage
- Fake relationship
- Love triangle
- Family pressure
- Revenge
- Public humiliation
- Sudden reversal of status
- Secret heir
- Misunderstanding
- Forbidden romance
- Final confession

### Use When
The story must keep audiences watching episode after episode.

### Agent Application
When applying Vertical Drama Formula:
- Start scenes late and leave early
- End almost every episode with a question, reveal, or threat
- Keep episodes emotionally simple but intense
- Use repeated reversals
- Make secrets collide in public settings
- Escalate every 1-3 episodes

### Output Template
```markdown
## Vertical Drama Episode Template

### Episode Number
[Episode #]

### Opening Hook
[Immediate emotional or visual grab.]

### Main Conflict
[The confrontation or problem.]

### Revelation
[New information that changes the situation.]

### Escalation
[How the problem gets worse.]

### Cliffhanger
[The unresolved question, shock, or danger.]
```

## 11. Seven-Point Story Structure

### Best For
Novels, fantasy, science fiction, adventure, long-form plotting.

### Core Idea
Start with the desired ending, then build a chain of turning points and pressure points that force the protagonist toward transformation.

### Seven Points
1. Hook
2. Plot Turn 1
3. Pinch Point 1
4. Midpoint
5. Pinch Point 2
6. Plot Turn 2
7. Resolution

### Use When
The user wants to plan a novel or long story efficiently.

### Agent Application
When applying Seven-Point Structure:
- Define the ending first
- Make the Hook show the opposite of the ending
- Use Pinch Points to show antagonistic pressure
- Use Plot Turns to change direction
- Use the Midpoint to shift the hero from reactive to active

### Output Template
```markdown
## Seven-Point Structure Outline

### Hook
[Where the protagonist begins.]

### Plot Turn 1
[The event that moves the story into the main conflict.]

### Pinch Point 1
[Antagonistic pressure shows the danger.]

### Midpoint
[The protagonist shifts from reaction to action.]

### Pinch Point 2
[The antagonist or problem strikes harder.]

### Plot Turn 2
[The final piece needed for resolution.]

### Resolution
[The ending and transformation.]
```

## 12. Fichtean Curve

### Best For
Thrillers, horror, action, suspense, fast-paced serials, vertical dramas, survival stories.

### Core Idea
The story begins with conflict and escalates through repeated crises until the climax.

### Pattern
1. Immediate crisis
2. Brief response
3. Bigger crisis
4. Brief response
5. Larger crisis
6. Emotional low point
7. Final crisis
8. Climax
9. Resolution

### Use When
The story needs momentum and constant tension.

### Agent Application
When applying Fichtean Curve:
- Open with trouble
- Avoid long exposition
- Use each crisis to reveal character
- Make each crisis worse than the last
- Allow only brief moments of relief
- Drive toward a peak confrontation

### Output Template
```markdown
## Fichtean Curve Outline

### Crisis 1
[The story opens with immediate trouble.]

### Response
[How the protagonist reacts.]

### Crisis 2
[The problem grows worse.]

### Response
[New attempt to solve it.]

### Crisis 3
[The stakes increase.]

### Emotional Low Point
[The protagonist nearly breaks.]

### Final Crisis
[The biggest danger arrives.]

### Climax
[The decisive confrontation.]

### Resolution
[The aftermath.]
```

# Hybrid Structure Recipes

## Commercial Fantasy Romance
Use:
1. Hero's Journey for the main adventure
2. Romance Structure for the relationship
3. Save the Cat for pacing
4. Fichtean Curve for danger escalation

### Formula
A heroine enters a magical world, falls for someone dangerous, discovers a secret, loses everything, then chooses love and power in the finale.

---

## Supernatural Shonen Romance
Use:
1. Shonen Battle Manga Structure
2. Hero's Journey
3. Romance Structure
4. Soap Opera cliffhangers

### Formula
A young hero discovers hidden power, trains under pressure, competes with a rival, falls into a love triangle, and faces a supernatural enemy tied to their destiny.

---

## Vertical Revenge Romance
Use:
1. Soap Opera / Vertical Drama Formula
2. Romance Structure
3. Fichtean Curve
4. Save the Cat

### Formula
A betrayed heroine loses everything, enters a fake relationship or hidden identity situation, humiliates her enemies one by one, and discovers the love interest has a secret connection to her past.

---

## Mystery Thriller
Use:
1. 12-Step Mystery Formula
2. Fichtean Curve
3. Three-Act Structure

### Formula
A disturbing mystery pulls the protagonist into danger, each clue creates a bigger threat, a false solution fails, and the final reveal exposes the hidden truth.

---

## Slice of Life Anime Short
Use:
1. Kishotenketsu
2. Story Circle

### Formula
A simple situation develops gently, a twist reframes it emotionally, and the ending reveals a quiet truth about the character.

# Agent Behavior Rules

## When the User Gives a Story Idea
The agent should:
1. Identify the genre and format
2. Recommend 1-3 structures
3. Explain why those structures fit
4. Offer a hybrid structure if appropriate
5. Create a beat sheet using the chosen structure

## When the User Asks for an Outline
The agent should:
1. Ask for or infer length and format
2. Choose the best structure
3. Create major acts or arcs
4. Break into beats
5. Add emotional turns and cliffhangers if serialized

## When the User Asks to Improve a Story
The agent should check:
- Is the protagonist's goal clear?
- Is there an inciting incident?
- Are the stakes escalating?
- Is there a midpoint reversal?
- Is there an emotional low point?
- Does the climax resolve the theme?
- Are secrets and reveals placed effectively?
- Does each episode or chapter end with momentum?

## When the User Wants a Microdrama
The agent should prioritize:
- Immediate hook
- High emotional stakes
- Short scenes
- Public confrontations
- Secrets
- Betrayals
- Reversals
- Cliffhangers
- Love triangle or forbidden desire
- Clear episode-by-episode escalation

## When the User Wants Manga or Anime Structure
The agent should prioritize:
- Strong character desire
- Iconic rivals
- Distinct power system
- Training or discovery phase
- Escalating enemies
- Emotional battles
- Friendship, sacrifice, and transformation
- Arc-ending reveal of a bigger threat

# Diagnostic Questions

Use these only when needed.

1. What format is this story: novel, screenplay, manga, webtoon, game, microdrama, or series?
2. What genre is it?
3. Who is the protagonist?
4. What do they want?
5. What do they fear?
6. What secret or wound drives the story?
7. What is the central conflict?
8. What is the romantic or emotional engine?
9. What is the biggest twist?
10. What should the audience feel at the end?

# Final Instruction to Agent

Always remember:

A story structure is not the story.  
The story comes from character desire, emotional stakes, conflict, secrets, choices, and consequences.

Use structure to make the story clearer, stronger, and more emotionally addictive.
