# Seedance 2.0 Video Prompt Generator

Use this skill to convert story material into clear, compact, production-ready prompts for Seedance 2.0.

Before generating Seedance prompts, read [references/full-guide.md](references/full-guide.md) and follow its prompt formula, reference limits, timestamp pacing, dialogue-scene workflow, audio guidance, exclusions, and output format.

For this project:

- Treat `Core/MyCatBoyfriendDeamon_Script.txt` as the source for written episodes once episode scripting begins.
- Preserve continuity with the character bible, creature bible, worldbuilding file, and arc outline.
- Convert each episode into Seedance-friendly visual beats rather than prose summary.
- Keep vertical microdrama pacing in mind: readable action, strong emotion, clear staging, and final hook images.
- Include character appearance, location, lighting, atmosphere, magical effects, creature movement, camera framing, dialogue, ambience, music, and useful exclusions when relevant.
- Use `9:16` as the default aspect ratio for this project unless the user says otherwise.
